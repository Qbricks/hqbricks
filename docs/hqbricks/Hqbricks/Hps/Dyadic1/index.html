Module Hps.Dyadic1

Dyadic modulo 1.

This module implements Dyadic modulo 1, which involves rationals of the form:

\frac{num}{2^{den\_pow}} \mod 1

where num is Zarith big integer of type Z.t representing the numerator and den_pow is an int representing the power of 2 of the denominator.
